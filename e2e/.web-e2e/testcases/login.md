# 登录模块 - 测试用例

> 模块：登录 (Login) · 角色：未登录访客 · 覆盖版本：MVP v1

## 模块说明

登录页承载"按角色进入不同后台"的核心分流，共 4 个角色 Tab：
- **总部端 (admin)** → 工作台概览
- **分校端 (campus_admin)** → 教务组织
- **教师端 (teacher)** → 今日教学
- **学员端 (student)** → 学生首页

关键元素：
- 角色 Tab × 4
- 用户名输入（placeholder = `输入工号或账号`）
- 密码输入（placeholder = `请输入登录密码`）
- 登录按钮
- 忘记密码? 入口
